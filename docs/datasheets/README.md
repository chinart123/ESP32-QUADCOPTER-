# Component Datasheets

Drop the PDF datasheets for the project's electrical devices here, so the hardware reference
lives alongside the code.

Suggested contents:

- **ESP32-S3** (SuperMini module / SoC)
- **MPU6050** — 6-axis IMU
- **VL53L1X** — Time-of-Flight ranging sensor
- **PMW3901** — optical-flow sensor
- **Motors / propellers** — spec sheets
- **Battery / ESC** — 1S cell + driver specs

See the main [`README.md`](../../README.md) for how each device is wired.
